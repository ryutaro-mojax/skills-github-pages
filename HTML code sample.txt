<h2 style="border-left: 6px solid #f06292; padding-left: 10px;"><id="anker1">副業を始める前に知っておきたい3つのこと</h2>

<h3 id="anker4">目的を明確にする</h3>
<p>まず初めにあなたが副業を始めようと思った理由を明確にしましょう。</p>
<p>収入を増やして生活の質を向上させたり、子供の学費を確保したりなどでしょうか？</p>
<p>副業を通してスキルアップを目指し、本業等でさらなる飛躍を求めているからでしょうか？</p>
<p>どんな理由でも構いません。あなたが副業を必要と考える理由を明確にしましょう。</p>
<p>副業を始めるのは簡単そうに見えますが、PCの準備など少なくない初期投資が必要だったり、続けるのが思っていたより難しかったということがあります。<br>副業を始めたけれど、やめてしまった理由には、</p> 
<div style="border:4px double #ff6699; padding: 20px; margin: 20px 0;">

<ol>
 <li><strong>時間の管理</strong><br>本業が終わったあとに副業に費やせる時間の確保が難しかった</li> 
 <li><strong>体力・精神力の管理</strong><br>本業が終わったあとに副業に取り組める体力・精神力の確保が難しかった</li> 
 <li><strong>収入の壁</strong><br>副業をしている人の多くは月１万円の報酬を越えるのが難しい。成果が出るまで学び続ける覚悟足りなかった</li> 
</ol>
</div>

<p>と続きます。</p>
<p>これらは副業を始めると必ずぶつかる、最初の壁と言えます。</p> 
<p>成果が出る前に副業をやめてしまっては、かけたお金や時間が無駄になります。</p>
<p>心理学では、<strong>具体的で、測定・達成可能なゴールを設定することで、モチベーションや集中力が維持できる</strong>とされており（目的設定理論）、副業で続け成果を出し続け、消費した費用（お金や時間などのリソース）から対効果を得るためには、明確な目的を設定することで最大限の効果を得られます。 
<h3 id="anker5">時間の使い方を見直す</h3>
<p>副業を始めることは簡単そうに見えますが、案件を獲得するまでに、必要なスキルの学習や、案件獲得後にはミーティングや実務作業など、少なくない時間が必要です。ところが、副業初心者が陥りがちな問題に、<strong>時間管理</strong>の意識の低さがあります。</p>
<p>1日は24時間しかありません。その中でどれくらいの時間が副業に充てられるかを把握する必要があります。また無理な時間設定で、本業に支障をきたしたり、続ける体力を失い副業をやめてしまったりしては本末転倒です。</p>
<p>１日の時間を整理し、自身が副業に充てられる時間を確保するようにしましょう。</p>
<div style="border: 4px double #ff6699; padding: 20px; margin: 20px 0;">

<ul>
 <li>朝起きる時間</li>
 <li>通勤中</li>
 <li>休憩時間</li>
 <li>帰宅後の時間</li>
 <li>子供を寝かしつけたあとの時間</li>
</ul>
</div>

<p>上の例を参考にどのタイミングで、どれくらい時間を確保できるのかを具体的にまとめることが、無理なく副業を続けるコツです。</p>
<h3 id="anker6">家族の理解を得る</h3>
<p>副業を始めるにあたってもう一つ考える必要があるのが、<strong>家族の理解</strong>です。前章までで述べてきたように、副業には多くの時間と労力を要します。それらを確保するには必然的に家族の理解が必要でしょう。</p> 
<p>それだけでなく、家族に内容を詳しく伝えずに副業を始めてしまうと様々な軋轢が生まれます。</p> 
<ul>
 <li>本業が疎かになり、収入に影響しないか</li>
 <li>過労や睡眠不足など健康面は大丈夫か</li>
 <li>家族の時間は確保できるのか</li>
 <li>初期投資に係る費用の回収や補填はできるのか</li>
 <li>本当に副業が必要なのか</li>
</ul>
<p>など、あなたが内容を伝えていないことで、家族が不安を抱きます。自分のモチベーションの維持と合わせて、家族の理解を得ることで、副業を続けられる環境を整えましょう</p>  
<p>次に、具体的にどのように理解を得るかの例を挙げます。</p>
<div style="border: 4px double #ff6699; padding: 20px; margin: 20px 0;">



<ol>
 <li><strong>本業が疎かになり、収入に影響しないか</strong><br>2023年のパーソナル総合研究所の調査では、副業を行った人を測定したところ、多くのポジティブな効果を得ているということがわかりました。以下のように副業を行うことは本業へポジティブな効果があることを説明しましょう。</li> 
 <table border="1">
  <tr>
   <th><strong>副業の効果</strong></th>  
   <td>視野の拡大・スキル、知識の向上・本業へのモチベーションや就業意識の向上など</td> 
  </tr>
  <tr>
   <th><strong>就業力の向上</strong></th> 
   <td>傾聴力・発信力・状況把握能力などのエンプロイアビリティの向上</td>
  </tr>
</table>
 <li><strong>過労や睡眠不足など健康面や、家族の時間の確保は大丈夫か</strong><br>明確化した時間の管理を家族と共有することで、本業・副業・家族の時間と休息の時間が見える化され、不安が解消されるでしょう。</li>
 <li><strong>初期投資に係る費用の回収や補填はできるのか・本当に副業が必要なのか。</strong><br>成果が出るまでは時間がかかるかもしれませんが、例えば当月にどれくらいの報酬が得られたのかを、細かく家族と共有するなどして不安を解消しましょう。また、報酬が一定額に達したら旅行に行くなどの計画を立てることで、家族と一緒に副業を続けていける環境が整うでしょう。</li> 
</ol>
</div>

<h2 style="border-left: 6px solid #f06292; padding-left: 10px;"> <id="anker2">初心者でも安心！副業の始め方ステップ３</h2>  

 <h3 id="anker7">1.自分にあった副業を探す</h3> 
<p>副業を始めるにあたり、自分にあった副業を始めるということが成果を出すうえで重要です。</p> <p>そのためにも、まずは自分にできること、既に得ているノウハウや技術、数値化できる実績をリスト化してみることをおすすめします。</p> 
<p>また、副業を始めて、成果を出すまで努力を続けるために、自分が夢中になれるものについて考えるのも良いでしょう。</p> 
<p>次に、クラウドソーシングサイトに登録し、実際の案件を検索してみましょう。<br>募集のページには必要な知識や求められる技術について記載されています。リストと照合してみて自分にあっているか、やってみたいと思うかを判断するのも良いでしょう。</p>
 <div style="border: 4px double #ff6699;font-size: 100%; padding: 20px;">  
<ul>
 <li>知識や技術をリスト化</li> 
 <li>数値化できる実績をリスト化</li>
 <li>夢中になれるものを考える</li>  
 <li>クラウドソーシングサイトで実際の案件を検索</li> 
</ul> 
</div> 
<h3 id="anker8">2.小さく始めてみる</h3> 
<p>副業を始めるには、少なくない初期投資が必要です。また学習にも時間をかける必要があります。そのためまずはハードルを低く設定し、小さく始めることが副業を続けるコツです。</p> 
<h3 id="anker9">3.実績を積んで単価アップを狙う</h3> 
<p>さらに、小さな案件でも、忍耐強く実績を積み重ねながら、同時に自身の技術を磨くことで、単価アップをねらっていきましょう。</p>   
 <h2 style="border-left: 6px solid #f06292; padding-left: 10px;"><id="anker3">副業を始めるときの注意点</h2>
 <p>税金・確定申告・就業規則など見落としやすい点に注意しましょう。</p> 
<p>その年の1月から12月までの副業による収入が20万円を超える場合は確定申告が必要です。故意に確定申告を行わなければペナルティを科される可能性があるので注意しましょう。</p>
<p>また本業の就業規則で副業を行っても良いかどうかを確認し、トラブルを回避しましょう。</p>
